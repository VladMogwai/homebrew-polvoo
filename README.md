> Like Docker Desktop — but for your local dev projects.

## Install
```bash
brew tap VladMogwai/dev-dashboard
brew install --cask dev-dashboard
```

## What is Dev Dashboard?

Dev Dashboard is a macOS utility for developers to manage all their local projects in one place — start/stop processes, view logs, manage terminals, track git branches and more.

## Requirements

- macOS 10.12+
- Apple Silicon or Intel Mac

## Uninstall
```bash
brew uninstall --cask dev-dashboard
brew untap VladMogwai/dev-dashboard
```
